# FarsiGo Roadmap

This roadmap describes the planned evolution of FarsiGo.

The goal is not to build a perfect language from the beginning.

The goal is to evolve a simple and consistent form of Persian through experimentation and community feedback.

---

# Version 0.1

Foundation

- Project website
- Manifesto
- Philosophy
- Core grammar
- Basic examples

# Version 0.2

---

# Long-Term Vision

FarsiGo aims to become:

- A simplified fork of Persian
- Easy to learn
- Easy to write
- Easy to type
- Easy for AI
- Easy for education
- Easy for global communication
