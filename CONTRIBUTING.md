# Contributing to FarsiGo

Thank you for your interest in FarsiGo!

FarsiGo is an open-source initiative to create a simplified, learnable, and globally accessible form of Persian while preserving its identity.

## How to contribute

You can contribute by:

- Improving the grammar
- Suggesting new rules
- Reporting inconsistencies
- Writing examples and stories
- Improving documentation
- Fixing typos
- Translating documentation

## Design Principles

Every proposal should respect these principles:

- Simplicity first
- Minimal grammar
- Consistency
- Easy to learn
- Easy to type
- Compatible with Persian

## Pull Requests

Before submitting a Pull Request:

1. Explain the problem.
2. Explain your proposed solution.
3. Provide examples.
4. Keep changes as small as possible.

Thank you for helping improve FarsiGo.
