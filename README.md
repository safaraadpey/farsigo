# FarsiGo

فارسی میانبر
Farsi Mianbor

FarsiGo is an open-source initiative to create a simplified, learnable, and globally accessible form of Persian while preserving its identity.

## Goals

- Easier to learn
- Easier to write
- Easier to type
- Easier to teach
- Compatible with Persian

## Quick Example

**Standard Persian**

من دارم غذا می‌خورم.

*man daram ghaza mikhoram.*

**FarsiGo**

من  غذا می خوردن.

*man ghaza mi khordan.*
