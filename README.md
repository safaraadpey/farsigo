# FarsiGo

فارسی میانبر
Farsi Mianbor

FarsiGo is an AI-assisted simplified fork of Persian for learning, writing, and communication.
## Goals

- Easier to learn
- Easier to write
- Easier to type
- Easier to teach
- Compatible with Persian

## Quick Example

**Standard Persian**

من دارم غذا می‌خورم.

*man daram ghaza mikhoram.*

**FarsiGo**

من  غذا می خوردن.

*man ghaza mi khordan.*
